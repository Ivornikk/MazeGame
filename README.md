## Overview

This repository contains a Labyrinth game developed by **team RAD**.

The game is written in **C++** and uses **raylib** for graphics, audio and input handling. The project was developed with Visual Studio 2022 and is hosted on GitHub for version control and collaboration.

## Features

- Maze generation and rendering
- Player movement and collision
- Simple UI and controls using raylib
- Save / load level (optional)

## Tools & Technologies

### C++

Main programming language used for game logic.

### raylib

Lightweight library for graphics, audio, and input handling.

### Visual Studio 2022 (MSVC)

Development environment and compiler used during development.

### GitHub

Repository hosting, issue tracking, and collaboration.
